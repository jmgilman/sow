# GitHub Issue Integration Workflow

## Intent

This PR implements the ability to create sow projects directly from GitHub issues labeled with 'sow'. Users can select an issue from the wizard, and sow will automatically create a linked branch, preserve issue context, and launch Claude with the issue details. This enhances the workflow by seamlessly transitioning from issue triage to structured implementation.
